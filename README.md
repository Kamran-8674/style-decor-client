# StyleDecor

# Features

- Users can book home and ceremony decoration services online.
- Users can explore different decoration packages with detailed information.
- Users can apply to become a decorator through the platform.
- Admin can manage services, bookings, decorators, and users.
- Decorators can view assigned projects and update project status step-by-step.
- Secure authentication and protected routes are implemented using Firebase Authentication and JWT.
- Stripe payment integration allows users to pay for bookings securely.
- Dynamic search functionality for services.
- Responsive modern UI built with DaisyUI and TailwindCSS.
- Service coverage map implemented using React Leaflet.
